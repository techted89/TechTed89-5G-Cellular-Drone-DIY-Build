# TechTed89-5G-Cellular-Drone-DIY-Build
Open-source 5G drone flight controller &amp; telemetry server. Uses a split-brain architecture (Linux SBC + FC) routed through a Tailscale VPS mesh to bypass CGNAT. Delivers ultra-low latency WebRTC video, duplex MAVLink control, dynamic Return-to-Home failsafes, and SMS emergency beacons. Built for BVLOS flight and cross-platform API control.
